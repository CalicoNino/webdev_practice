Important Terminologies with Definitions

First developed by Tim Berners-Lee in 1990, HTML is short for Hypertext Markup Language.

Developed by Brendan Eich and originally known as LiveScript, the programming language JavaScript was renamed in 1995 and now a trademark of Oracle.

Short for cascading style sheets, CSS is a language used to describe reusable styles for presenting documents written in a markup language. Its concept was originated by Håkon Wium Lie in 1994.

Short for extensible markup language, XML is a specification developed by W3C starting with the recommendation on February 10, 1998.

Short for Asynchronous JavaScript and XML, AJAX is a term coined by adaptive path.

Introduction

Now that you know the basics of HTML5 and CSS3, you might be wondering how to do things like calculations, event handling, or creating a dynamic web page. This is where Javascript can improve your web pages greatly. This is not the same as Java which is object orientated. Javascript is a scripting language to enhance both the appearance and functionality of web pages.

Similar to CSS, you can have inline Javascript code within your HTML5 files or you can choose to have a seperate .js file and simply link it in your HTML5. In order to use Javascript inline you need to create a script element as follows (Deital p.187):

In that example, a line was simply written to the document/ web page. If you would rather use javascript in a seperate file than you simply need to add this to your HTML5 file.

<script src="nameOfFile.js" type="text/javascript"></script>

This doesn't necessarily need to go into the head of your file. Since your web browser will chronologically process the HTML5 and script in the order it is presented, if you have 'global' function or reference elements that have not yet been processed than the browser will throw an error. It doesn't know that the element you are referencing is below the script element. It may be safe to put your script element below your HTML5 content or try to design a system where this wouldnt be an issue.

There are a few basics that we will be covering in this tutorial which will include how to common knowledge with Javascript, call functions, and trigger events

Common Javascript

There are a few main objects and methods that people just starting out with Javascript should know. Before listing some of them, it should be noted that Javascript behaves similarly to python in that it does not explicitly define an element's/object's type. You do not need to specifiy the parameter type (ie. function(nameOfParam){} rather than function(int name){}) or need to specifying one for a variable. You can simply give the type 'var' for every variable you create (similar to the capabilities of C#).

With that in mind, here is a list of some common Javascript knowledge:

  • Document: the 'document' object represents the HTML5 document object. This gives you access to all of the elements found in the file. These are known as the Document Object Model (DOM) which we will later discuss how to get elements and attributes for these objects.
  • Methods of DOM: Similar to the concept of a class object, DOM objects have attributes and methods which can be called. For example if we want to write a line to the document, the method 'writeln()' can be called.

    • document.writeln("Hello World!")

  • Attributes of DOM: Objects have attributes that can be set or simply just to get the value of an attribute. You can call the method '.setattribute(...)' or to modify/get CSS attributes you can directly set them. For example if you want to hide an element you can do the following

    • myClass.style.display = 'none';

  • Semi-colon: The semi colon is not required but it is greatly advised to reduce any bugs that may occur
  • Case Sensitive: Javascript is case sensitive so be sure to have the correct casing!
  • Quotes You can mix and match single quotes ' with double quotes " and they should work the same in any case. What is useful is when you want to display either single/ double quotes to the user. You should then nest the quotes you wish to show in the opposing quotes.
  • Window: the window object can be thought of as the browser window. You can invoke an alert dialog with window.alert("Warning!") or create a function that will be called when the window loads window.onload() = function() {}; We will discuss this syntax later in the tutorial.
    • You can also get user input with a prompt dialog (window.prompt("Please enter a number");)
  • Concatinating + Adding: the '+' sign can be used to concatinate strings as well as add numbers. You must be careful with this and not confuse concatinating string with adding numbers. Look at the following as an example, where y = 5.

    • "y + 2 =" + y + 2

    Expected: "y + 2 = 7"

    Result: "y + 2 = 52"

    • this is the wrong result since we mixed string with integer values. We need to specify exactly that we want to add two integers by grouping them within brackets. This will add (y + 2) as ints then concat them with the "y + 2 =" string.

    • "y + 2 =" + (y + 2)

    Result: "y + 2 = 7"

  • Strict Equals/Not Equals: Typically it is good enough to compare two objects/values with a == a or a != b. But in Javascript the following condition would also return true, "72" == 72, even though one is a string and another is an integer. To strictly compare you must use '===' or '!=='

    • "72" === 72 will result in false

  • If Condition: you can use if, else if, and else as you would with any other programming language. if(condition){}else if(condition){} else{};
  • While Loops: you can loop until a condition is met with while(condition){};
  • For Loops:a for loop allows you to iterate through an indexed array or list, for(...);
  • Increment, Decrement: For integers, you can easilly increment or decrement them with the following (Deital p.239)

Functions

You can create functions in Javascript that will help you group together common code with parameters. For example the basic structure is: function AddTwo(input) { return input+2; }

You can also use common function such as many Math functions. To list a few:

  • Math.pow(2,3) is 2 to the power of 3
  • Math.random() returns a random number
  • Math.floor(2.33), Math.ceil(2.99), Math.round(2.5) will round your values

You can retrieve HTML5 elements by their id, class, etc.

  • ID: you can use document.getElementById("idVal") or using jquery $("#idVal")
  • Class: you can use document.getElementsByClassName("classVal") or using jquery $(".classVal")

You can also have recursive functions (function that call itself) or iterative functions.

Event Handling

Buttons, forms, inputs, and various other HTML5 elements can trigger events! Javascript allows you to control what happens when they are triggered. You can either use a inline functions, global elements, or use JQuery to invoke these methods.

Inline: in your HTML5 element you can set the event method to call a javascript method you have written.

Global: you can retrieve the element and then use their event handling method in a global way

JQuery: if you use JQuery, you can use a shorter form than having to globally retrieve the element first

HTML5 Canvas

The canvas element is used to create 2D graphics, animation, images, videos. This is where you can let your creativity shine with more artistic elements.

The canvas works with a grid system of (x horizontal, y vertical). top left corner has the grid value of (0,0). Moving right will increase the x value (ie. (50,0) is 50 pixels to the right of the top left corner). Moving down will increase the y coordinate (ie. (50,10) is 50 px right and 10 px down from the left corner).
